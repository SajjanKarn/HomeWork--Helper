# HomeWork--Helper
This is just a GUI software that will help u in your Homeworks! it can help u in physics,biology,chemistry,astronomy,computer Science, GK and many more!

========================================== HOW TO USE THIS APPLICATION ==============================================================
1.  First type the topic name in Find Anything Box that u want to search!
2.  Then just click on Find Answer Button then ur answer will appear in Text Box
3.  You can use clear buttons for clearing or deleting answers
3.  You can also use Speak button for speaking answers!
